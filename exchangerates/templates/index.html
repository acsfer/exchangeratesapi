Foreign exchange rates API
with currency conversion

Exchange rates API is a free service for current and historical foreign exchange rates
published by the European Central Bank 

            
GET /api/latest HTTP/1.1

Usage

Get the latest foreign exchange reference rates. 

            
GET /api/latest HTTP/1.1

Get historical rates for any day since 1999. 

            
GET /api/2010-01-12 HTTP/1.1

Rates are quoted against the Euro by default. Quote against a different currency by setting the base parameter in your request. 

            
GET /api/latest?base=USD HTTP/1.1

Request specific exchange rates by setting the symbols parameter. 

            
GET /api/latest?symbols=USD,GBP HTTP/1.1

Please cache results whenever possible this will allow us to keep the service without any rate limits or api key requirements.
